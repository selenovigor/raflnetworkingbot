# Patterns

## Code

- Минимальная логика без лишних абстракций.
- Конфигурация только через environment variables.
- Секреты не хранятся в коде.
- Ответы Telegram форматируются через HTML parse mode.

## Git

- `main` - основная ветка.
- `.env` запрещен к коммиту.
- `.env.example` должен отражать актуальный набор переменных.

## Verification

- Синтаксис: `python3 -m py_compile bot.py`.
- Manual QA: сообщения `1`, `2`, `3`, `4`, `5`, произвольный текст, сообщение под другим постом.
