# Project

## Name

`raflnetworkingbot`

## Audience

Аудитория Telegram-канала `@rafl_work`: креаторы и специалисты в мире футбола и спорта.

## Goal

Бот отвечает в комментариях под конкретным постом канала. Пользователь пишет цифру `1`-`5`, бот отвечает персональным текстом. На любой другой ввод бот отвечает отбивкой.

На каждую цифру есть несколько вариантов ответа; бот выбирает один случайно. На каждый третий валидный цифровой запрос пользователя в треде бот добавляет напоминание подписаться на проекты `@rafl_work` и `@raflmedia`.

## Current Status

- Локальная разработка завершена.
- Тестовый прогон в тестовом канале пройден.
- Код опубликован в GitHub: `selenovigor/raflnetworkingbot`.
- Деплой на Render отложен из-за цены за отдельный background worker.
- Был выбран VPS-подход, чтобы в будущем держать несколько простых ботов на одном сервере.
- Попытка деплоя на JustHost VPS заблокирована: SSH-порт открыт, но соединение закрывается до запроса пароля; Proxmox/VPS-консоль работает нестабильно.
- Production deploy выполнен на Timeweb VPS `81.200.145.88`.
- Бот запущен через `systemd` service `raflnetworkingbot` и продолжает работать при закрытом Mac.
- Рабочий Telegram-тест после VPS-деплоя пройден.
