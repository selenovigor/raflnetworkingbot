# Project

## Name

`raflnetworkingbot`

## Audience

Аудитория Telegram-канала `@rafl_work`: креаторы и специалисты в мире футбола и спорта.

## Goal

Бот отвечает в комментариях под конкретным постом канала. Пользователь пишет цифру `1`-`5`, бот отвечает персональным текстом. На любой другой ввод бот отвечает отбивкой.

## Current Status

- Локальная разработка завершена.
- Тестовый прогон в тестовом канале пройден.
- Код опубликован в GitHub: `selenovigor/raflnetworkingbot`.
- VPS заказан, ожидаются доступы.

## Non-goals

- База данных.
- Админ-панель.
- Сбор персональных данных.
- Ответы под несколькими постами в первой версии.
