# User Spec: comment-reply-bot

Status: implemented, pending VPS deploy

## 1. Summary

Telegram-бот отвечает в комментариях под одним конкретным постом канала. Пользователь выбирает цифру на картинке, пишет `1`-`5`, получает персональный текст.

## 2. User and Problem

| Вопрос | Ответ |
| --- | --- |
| Кто пользователь? | Подписчик канала `@rafl_work`, интересующийся креативной работой в футболе |
| Какая проблема? | Нужно быстро отдать персональный результат по выбранной цифре в комментариях |
| Как пользователь решает это сейчас? | Вручную ждет ответ или читает пост без интерактива |
| Что изменится после фичи? | Пользователь получает мгновенный reply от бота |

## 3. Goals

- Реагировать только на комментарии под одним постом.
- Отвечать реплаем на сообщение пользователя.
- Поддерживать варианты `1`-`5`.
- Давать отбивку на любой другой текст.
- Не реагировать на собственные сообщения.

## 4. Non-goals

- Несколько активных постов.
- Статистика ответов.
- База данных.
- Админка.

## 5. User Scenarios

### Scenario 1: Valid number

1. Пользователь открывает комментарии под тестовым постом.
2. Пользователь пишет `3`.
3. Бот отвечает реплаем с текстом для варианта 3.

### Scenario 2: Invalid input

1. Пользователь пишет `привет`.
2. Бот отвечает реплаем с отбивкой про цифры `1`-`5`.

### Scenario 3: Other post

1. Пользователь пишет под другим постом.
2. Бот ничего не отвечает.

## 6. Acceptance Criteria

- [x] `1` возвращает текст 1.
- [x] `2` возвращает текст 2.
- [x] `3` возвращает текст 3.
- [x] `4` возвращает текст 4.
- [x] `5` возвращает текст 5.
- [x] Другой текст возвращает отбивку.
- [x] Ответ отправляется reply.
- [x] Бот ограничен `POST_ID` и `TARGET_CHAT_ID`.
- [x] Локальный тестовый прогон пройден.
- [ ] VPS deploy пройден.

## 7. Risks

| Риск | Вероятность | Влияние | Как снижаем |
| --- | --- | --- | --- |
| Токен раскрыт | Высокая | Высокое | После финального теста сделать `/revoke` и заменить токен |
| Два polling-процесса | Средняя | Среднее | Перед VPS запуском остановить локальный процесс |
| Неверный POST_ID в бою | Средняя | Среднее | Включать `DEBUG_UPDATES=true` для поиска ID |
